# ⚛️ React Core Concept Questions

> A structured list of important React conceptual questions.

---

## 📌 1️⃣ What is JSX, and why is it used?
> JSX is JavaScript XML. We can write html, css, js in a single file of JSX in module system. It is used to make web interactive. We used it in the libaray of JavaScript most popular React. 

---

## 📌 2️⃣ What is the difference between State and Props?
> State is component memory we can store dynamic value and function inside it . Props are read only for passing or sharing data from one component to another component. 

---

## 📌 3️⃣ What is the useState hook, and how does it work?
> useState hook is a popular hook of react to add interactivity in websites. If we need to change anything multiple places of a website we can change that thing multiple places easily by using useState hook variable and setter function. 
---

## 📌 4️⃣ How can you share state between components in React?
>We can share data easily between components by using props. It is read only. 
---

## 📌 5️⃣ How is event handling done in React?
> Event handling done in react almost same as we do in html . First we need to declare a handleEvent function and then we need to use that function on a button via onClick attribute.
